1. Introduction

Gestion Morgue est une application de bureau conçue pour faciliter la gestion des défunts au sein d'une morgue ou d'un établissement funéraire. Elle permet de :

  • Enregistrer et suivre les défunts accueillis
  • Gérer les emplacements de stockage (tiroirs, chambres froides)
  • Planifier et tracer les interventions (thanatopraxie, soins)
  • Gérer les autorisations de sortie
  • Suivre les contacts familiaux
  • Générer des rapports et statistiques
  • Assurer la traçabilité complète via un journal d'audit

2. Connexion et premier accès

2.1 Écran de connexion

Lancez l'application. L'écran de connexion apparaît avec les champs :

  • Nom d'utilisateur — votre identifiant
  • Mot de passe — votre mot de passe confidentiel

2.2 Comptes par défaut

Lors de la première installation, quatre comptes sont créés automatiquement :

RôleIdentifiantMot de passePermissions
Administrateuradminadmin123Toutes les fonctionnalités
Médecinmedecinmedecin123Défunts, stockage, interventions, sorties, rapports, familles
Thanatopracteurthanatothanato123Défunts, stockage, interventions, sorties
Greffiergreffiergreffier123Défunts, rapports, journal, familles
Il est impératif de changer les mots de passe par défaut dès la première connexion.

3. Tableau de bord

Après connexion, le tableau de bord affiche un résumé de l'activité :

  • Défunts enregistrés — nombre total de défunts
  • Emplacements occupés — taux d'occupation du stockage
  • Interventions en cours — interventions planifiées ou en cours
  • Sorties du jour — autorisations de sortie récentes

Les données sont actualisées automatiquement toutes les 30 secondes.

4. Gestion des défunts

4.1 Liste des défunts

La vue Défunts présente la liste complète des défunts enregistrés, avec les colonnes :

  • N° de dossier
  • Nom et prénom
  • Date de naissance
  • Date de décès
  • Emplacement
  • Statut

4.2 Recherche et filtres

  • Recherche rapide — tapez un nom, un prénom ou un numéro de dossier dans le champ de recherche en haut
  • Filtres avancés — utilisez les filtres par date (période), statut, emplacement
  • Pagination — navigation par pages avec affichage du nombre de résultats

4.3 Ajouter un défunt

Cliquez sur Nouveau défunt. Remplissez les informations :

  • Identité (nom, prénom, date de naissance, sexe)
  • Informations sur le décès (date, lieu, cause)
  • Attribution d'un emplacement de stockage
  • Numéro de dossier (automatique ou manuel)

4.4 Modifier / Consulter

Double-cliquez sur un défunt dans la liste pour ouvrir la fiche détaillée. Vous pouvez :

  • Modifier les informations
  • Ajouter des pièces jointes
  • Ajouter des contacts familiaux
  • Consulter l'historique des interventions
  • Imprimer la fiche

5. Gestion du stockage

5.1 Vue d'ensemble

La vue Stockage affiche un plan des emplacements (tiroirs, chambres froides). Chaque emplacement est représenté par une carte indiquant :

  • Le code (ex: A-01)
  • L'état (libre / occupé)
  • La température
  • Le défunt occupant (si occupé)

5.2 Attribution

Lors de l'enregistrement d'un défunt, un emplacement libre peut lui être attribué. Les emplacements sont automatiquement marqués comme occupés.

5.3 Libération

Lors d'une sortie, l'emplacement est automatiquement libéré. Vous pouvez aussi libérer manuellement un emplacement depuis la vue Stockage.

6. Gestion des interventions

6.1 Types d'interventions

  • Soins de conservation — thanatopraxie
  • Toilette mortuaire
  • Soins esthétiques
  • Autopsie
  • Autre — intervention personnalisée

6.2 Planification

Créez une intervention en sélectionnant :

  • Le défunt concerné
  • Le type d'intervention
  • La date et l'heure prévues
  • Le technicien responsable
  • Les observations éventuelles

6.3 Suivi

Les interventions peuvent être marquées comme :

  • Planifiée — intervention à venir
  • En cours — intervention en réalisation
  • Terminée — intervention achevée
  • Annulée — intervention annulée

7. Autorisations de sortie

7.1 Demande de sortie

Pour autoriser la sortie d'un défunt :

  1. Sélectionnez le défunt concerné
  2. Indiquez la destination (funérarium, cimetière, crématorium, etc.)
  3. Précisez la date et l'heure de sortie
  4. Ajoutez les observations nécessaires
  5. Validez l'autorisation

L'autorisation est horodatée et signée électroniquement par l'utilisateur connecté.

7.2 Historique

Consultez l'ensemble des sorties avec filtres par date, défunt ou statut. Chaque sortie peut être imprimée.

8. Contacts familiaux

Gérez les contacts des familles des défunts :

  • Ajoutez un ou plusieurs contacts par défunt
  • Informations : nom, prénom, lien de parenté, téléphone, email, adresse
  • Modifiez ou supprimez les contacts existants
  • Accès rapide depuis la fiche du défunt

9. Rapports et statistiques

9.1 Rapports disponibles

  • Rapport d'activité — synthèse des opérations sur une période
  • Statistiques mensuelles — nombre d'entrées, sorties, interventions
  • Taux d'occupation — utilisation des emplacements de stockage

9.2 Export

Les rapports peuvent être exportés au format PDF ou imprimés directement.

10. Journal d'audit

Toutes les actions effectuées dans l'application sont enregistrées dans le journal d'audit :

  • Connexions et déconnexions
  • Création, modification et suppression de défunts
  • Attribution d'emplacements
  • Interventions (création, changement de statut)
  • Autorisations de sortie
  • Modifications des utilisateurs

Le journal est horodaté et horodaté avec le nom de l'utilisateur et l'adresse IP.

11. Gestion des utilisateurs

Cette section est accessible uniquement aux administrateurs.

11.1 Créer un utilisateur

  • Nom d'utilisateur (unique)
  • Nom complet
  • Adresse email
  • Rôle (ADMIN, MEDECIN, THANATOPRACTEUR, GREFFIER)
  • Mot de passe temporaire

11.2 Modifier / Désactiver

  • Modification des informations et du rôle
  • Désactivation d'un compte (sans le supprimer)
  • Réinitialisation du mot de passe

12. Pièces jointes

Vous pouvez associer des fichiers aux dossiers des défunts :

  • Types supportés — documents PDF, images (JPG, PNG), feuilles de calcul
  • Signature numérique — les pièces jointes peuvent être signées numériquement (PKCS#12)
  • Stockage — les fichiers sont stockés dans ~/.gestionmorgue/attachments/

13. Personnalisation

13.1 Thème

Basculez entre le thème clair et le thème sombre via le bouton Thème dans l'en-tête.

13.2 Créateur de thèmes

Personnalisez les couleurs de l'interface via Thème > Créateur de thème :

  • Couleur principale (barre de navigation, boutons)
  • Couleur secondaire (surbrillances, accents)
  • Couleur de fond
  • Aperçu en temps réel

13.3 Langue

Changez la langue de l'interface entre Français et Anglais via le bouton de langue dans l'en-tête.

14. Paramètres

La vue Paramètres permet de configurer :

  • Vérification des mises à jour — activer/désactiver la vérification automatique
  • Vérification SSL — activer/désactiver lors des appels API REST
  • Informations sur la base — type de base de données, URL de connexion
  • À propos — version de l'application, version de la base

15. Impression

Depuis les vues Défunts, Stockage et Journal, cliquez sur le bouton Imprimer pour générer un document imprimable :

  • Fiche défunt avec toutes les informations
  • État des lieux de stockage
  • Journal d'audit filtré

L'impression utilise la boîte de dialogue système d'impression.

16. Mises à jour

16.1 Vérification automatique

Au démarrage de l'application, une vérification de mise à jour est effectuée automatiquement. Si une nouvelle version est disponible, une notification s'affiche.

16.2 Vérification manuelle

Cliquez sur le bouton Mise à jour dans l'en-tête pour vérifier manuellement.

16.3 Installation

  1. Si une mise à jour est disponible, cliquez sur Télécharger
  2. La progression du téléchargement est affichée
  3. Une fois téléchargée, l'application se ferme et s'installe automatiquement
  4. L'application redémarre avec la nouvelle version
